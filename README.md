# Snakey
A simple Snake Game in C++ with Raylib

# Features
- [x] Snake
- [x] Food
- [x] Main Menu
- [x] Settings
    - [x] Initial Length
    - [x] Tick Rate
    - [x] Snake Wrapping
    - [x] Keybinds
- [x] Pause Menu
    - [x] Resume
    - [x] Adjust Settings
    - [x] Restart
    - [x] Return to Main Menu
- [x] Fun
